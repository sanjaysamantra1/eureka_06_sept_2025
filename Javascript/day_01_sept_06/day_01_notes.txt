Web Development
===============
-Developing a Website

1. FrontEnd : User Interface ( Look & Feel )
        HTML : Content ( paragraph,image,form,table,list,audio,video )
        CSS : Styling ( border,background,width,height,margin,padding )
        Bootstrap : CSS Library (twitter) ( Pre-defined css classes, responsiveness )
        Javascript : Behaviour
        Angular/React : Javascript Framework ( RAD , CLI , Template System, Routing , Form Handling)
2. BackEnd :  Develop REST APIs ( Java , .net , PHP , Python , Nodejs , Go)
3. Database : Holds the data ( oracle , mysql , mongoDB )
