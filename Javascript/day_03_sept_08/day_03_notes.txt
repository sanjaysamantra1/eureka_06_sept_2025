DataTypes 
=========
-Type of data we are dealing with
2 types : primitive , non-primitive

primitives : 
    number  : 4 , 4.5 , -2.3
    string : 'sachin', '6'
    boolean : true / false
    undefined : undefined
    bigint 
    symbol
non-primitive : Object

-primitives are immutable, non-primitives are mutable
-primitives are compared by value , non-primitives are compared by address


undefined vs null
=================
-when a variable is declared without any value, javascript assigns 'undefined' value to taht variable.
-null is an intentional value assigned by developers.
-typeof 'undefined' is undefined, typeof null is object

Note: even though typeof 'null' is object, null is a primitive value
